# Real-Time Face Emotion Detection

## Description
A Python project that detects human emotions in real-time using facial expressions. 
Uses a trained CNN model and OpenCV for face detection. Provides a web interface with Flask.

## Features
- Real-time emotion detection (Happy, Sad, Angry, Surprise, Neutral, Disgust, Fear)
- Flask web interface with index.html
- Includes trained model (.h5)
- Requirements file for easy setup

## Files
- `main.py` – main script to run the app
- `requirements.txt` – Python dependencies
- `model.h5` – trained CNN model
- `templates/index.html` – web interface
